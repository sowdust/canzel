About this website

The file robots.txt is used to instruct search engines visiting a website on what to index and what to ignore.
Many media websites use it to "hide" certain news for different reasons (legal requirements, corporate or personal requests...).
But we know the Internet never forgets: by the means of this website, we are monitoring what online media outlets are removing.
The source code of this website is public and can be used to monitor robots.txt files for other purposes.
You can follow the Twitter account @canzelculture to keep track of the new robots.txt entries as they are found.
